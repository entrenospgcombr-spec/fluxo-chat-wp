Gerencie a integração com a API Mistral, crie fluxos de atendimento e adicione um chatbot flutuante ao seu site.

== Description ==

O Mistral Chatbot Manager é um plugin completo para WordPress que permite:

1. **Configuração da API Mistral**: Insira sua chave de API e teste a conexão diretamente no painel administrativo.
2. **Fluxo de Atendimento**: Crie e visualize fluxos de atendimento com diagramas interativos usando Mermaid.js.
3. **Diretrizes para Agentes de IA**: Defina regras e comportamentos para seus agentes de IA.
4. **Chatbot Flutuante**: Adicione um botão de chat flutuante no canto inferior direito do seu site.

== Installation ==

1. Faça upload da pasta do plugin para o diretório `/wp-content/plugins/`
2. Ative o plugin através do menu 'Plugins' no WordPress
3. Configure o plugin em 'Mistral Chatbot' no menu administrativo

== Frequently Asked Questions ==

= Como obtenho uma chave de API da Mistral? =
Você pode obter uma chave de API criando uma conta no site da Mistral AI e gerando uma chave em seu painel de desenvolvedor.

= Posso personalizar a aparência do chatbot? =
Sim! Você pode alterar a cor, posição e título do chatbot nas configurações do plugin.

= O plugin funciona com outros provedores de IA? =
Atualmente o plugin é otimizado para a API da Mistral, mas pode ser estendido para outros provedores.
